# 🎮 Mays Blog

我的个人 Steam 游戏博客，展示我的游戏收藏和游玩时长。

## ✨ 功能特点

- 📊 展示 Steam 游戏库和游玩统计
- 🎯 游戏时长排行
- 🕐 最近游玩的游戏
- 🔍 游戏搜索和筛选
- 📱 响应式设计，支持移动端
- 🌙 暗色主题，游戏风格 UI

## 🛠️ 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **TypeScript** - 类型安全
- **Pinia** - 状态管理
- **Vue Router** - 路由管理
- **Vite** - 构建工具
- **GitHub Pages** - 静态托管
- **GitHub Actions** - 自动化部署

## 🚀 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📦 部署

推送代码到 `main` 分支后，GitHub Actions 会自动：
1. 获取最新的 Steam 游戏数据
2. 构建项目
3. 部署到 GitHub Pages

数据每天北京时间 10:00 自动更新。
